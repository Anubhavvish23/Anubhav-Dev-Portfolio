How It Works

  • Document Referrer: Detects if the user came from linkedin.com or linked.in
  • URL Parameters: Detects ?ref=linkedin or ?utm_source=linkedin
  • Session Storage: Prevents showing the message multiple times in the same session
  • Auto-hide: Message disappears after 5 seconds
  • Confetti: Triggers a LinkedIn-themed confetti animation
